เลือกใช้ prompt ด้านล่างตามความเหมาะกับเครื่องมือที่คุณใช้:

## 📹 Prompt สำหรับ AI Video Generator (เช่น HeyGen, InVideo, CapCut)

```
สร้างวิดีโอพรีเซ็นต์เป็นภาษาไทย สำหรับแอปพพลิเคชันบริหารคลังสินค้า (Stock Management System)

**ชื่อแอปพ:** STOCKPRODUCT - ระบบบริหารคลังสินค้า

**เนื้อหาวิดีโอ:**

Scene 1 (0:00-0:10) - Hook
- หน้าจอ: "เบื่องบริหารสต๊อกสินค้า?"
- ภาพประกอบ: เจ้าของธุรกิจกำลังเครียด โทรศัพท์ คำนวณระวนงาน

Scene 2 (0:10-0:30) - ปัญหา
- หน้าจอ: "สินค้าหาย ไม่รู้เหลือเท่าไร ขายราคาที่ไม่รู้กำไร"
- ภาพประกอบ: ห้องคลังที่รกโกยว, สินค้าเกะกะกะ, แป้นกวน

Scene 3 (0:30-1:00) - แนะนำ
- หน้าจอ: "พบกับ STOCKPRODUCT - ระบบบริหารคลังสินค้า"
- ภาพประกอบ: UI แอปพที่สะอาน สวยงาม ใช้งานง่าย

Scene 4 (1:00-2:00) - Features
- หน้าจอ: "จัดการสินค้า | ตรวจสอบสต๊อก | วิเคราะห์ยอดขาย"
- ภาพประกอบ: Screenshot ของ Dashboard, Product list, Reports
- แสดง: เพิ่มสินค้า, รับ-จ่ายสต๊อก, กราฟยอดขาย

Scene 5 (2:00-2:30) - Benefits
- หน้าจอ: "รู้ทันที | จัดการง่าย | เช็คได้ทุกที่"
- ภาพประกอบ: ใช้งานบนมือถือ, dashboard สวยงาม

Scene 6 (2:30-3:00) - CTA
- หน้าจอ: "เริ่มต้นใช้งานวันนี้"
- ภาพประกอบ: Logo, Call to Action, QR Code/Website

**สไตล์:**
- Modern, Professional, Tech-focused
- สีหลัก: Blue, Green, Dark background (ตาม brand)
- ฟอนต์: สะอาน อ่านง่าย
- เพลง: Energetic, Trustworthy

**Voiceover:** พากย์ชาย/หญิง เสียงนุ่มนวล มั่นใจ เป็นธรรมชาติไทย
**Music:** Tech startup style, upbeat ไม่ดังเกินไป
**Duration:** 3 นาที

**Call to Action:** "ลองใช้ฟรีวันนี้"
```

## 📊 Prompt สำหรับ PowerPoint Generator (เช่น Gamma, Beautiful.ai, Canva)

```
สร้างสไลด์ไพรีเซ็นเทชันสำหรับแอปพบริหารคลังสินค้า (ภาษาไทย)

**Title Slide:**
- Title: STOCKPRODUCT
- Subtitle: ระบบบริหารคลังสินค้า
- Tagline: จัดการง่าย รู้ทันที บริหารได้
- Background: Modern tech gradient

**Slide 2: ปัญหาที่เจอของเจ้าของธุรกิจ**
- สินค้าหายได้ง่าย
- ไม่รู้สต๊อกคงเหลือ
- คำนวณราคาทำผิดพลาด
- รู้กำไรไม่ชัวร์
- Icon: Warning signs, messy warehouse

**Slide 3: แนะนำ**
- STOCKPRODUCT ตอบโจท์ทุกปัญหา
- จัดการสินค้าครบวงจน
- ตรวจสอบสต๊อกแบบ Real-time
- รายงานวิเคราะห์ชัดเจน
- Icon: Solution, checkmarks, happy business owner

**Slide 4: คุณสมบัติหลัก**
- หัวข้อ: ทุกสิ่งที่คุณต้องการ
- Bullet points:
  - ✅ จัดการสินค้า (CRUD)
  - ✅ รับ-จ่ายสต๊อก (IN/OUT/ADJUST)
  - ✅ Dashboard สรุปข้อมูล
  - ✅ รายงานยอดขาย & กำไร
  - ✅ วิเคราะห์ตามหมวดหมู่
  - ✅ แจ้งเตือนสต๊อกต่ำ
- Icon: Checklist, features

**Slide 5: Dashboard - ภาพรวม**
- Screenshot UI จริงของ Dashboard
- แสดง: สถิติ, กราฟกำไร, แจ้งเตือนสต๊อกต่ำ
- Caption: "ทุกข้อมูลสำคัญ ในหน้าเดียว"

**Slide 6: Reports - รายงานวิเคราะห์**
- Screenshots:
  - กราฟแนวโน้มยอดขาย
  - PieChart สัดส่วนหมวดหมู่
  - ตารางอันดับสินค้า
- Caption: "วิเคราะห์ข้อมูลลึกซึ้ง"

**Slide 7: Mobile-Friendly**
- Mockup แอปบนมือถือ
- แสดง Responsive Design
- Bullet points:
  - ใช้งานได้ทุกที่
  - Card layout บน mobile
  - Table layout บน desktop
  - Dark/Light mode

**Slide 8: ใครควรใช้**
- Target: เจ้าของธุรกิจ SME, ร้านค้า, ค้าส่ง
- Icon: Business owners, shops

**Slide 9: Pricing (ถ้ามี)**
- Free trial
- ราคาแพ็กเกจ
- Contact info

**Slide 10: CTA**
- "เริ่มต้นใช้งานวันนี้"
- QR Code / Website / Contact
- Social media

**Design Style:**
- Modern, Clean, Tech
- Primary color: Blue (#3B82F6)
- Secondary: Green (#10B981)
- Background: Dark or Light mode options
- Font: Sarabun หรือ Prompt (Google Fonts)
```

## 🎬 Prompt สำหรับ AI Presentation (สั้น กระชับ)

```
สร้างไพรีเซ็นเทชันแนะนำแอปพ "STOCKPRODUCT - ระบบบริหารคลังสินค้า"

เนื้อหา:
1. ปัญหา: เจ้าของ SME ปวดหัวเรื่องสต๊อก สินค้าหาย ไม่รู้กำไร
2. แนะนำ: STOCKPRODUCT - แอปบริหารคลังเพียรเพียร เข้าใจง่าย
3. Features: Dashboard, Product Management, Stock Control, Reports, Analytics
4. Benefits: ประหยัดเวลา ลดความผิดพลาด รู้ข้อมูลทันที
5. CTA: เข้าใช้งานฟรี

สไตล์: Modern tech, blue-green theme, professional, trustworthy
ภาษา: ไทย
เสียง: AI voiceover ภาษาไทย หรือ Instrumental หากไม่มี
เพลง: Tech startup upbeat
```

## 💡 เคล็ดลับ

**ถ้าใช้ AI Video Generator:**
- ใช้ prompt แบบ "Scene by Scene" จะได้ผลละเอียดกว่า
- ระบุความยาวชัดเจน (เช่น 3 นาที, 2 นาที)
- ขอให้ AI suggestions shots/close-ups ของ UI

**ถ้าใช้ PowerPoint Generator:**
- ให้ระบุจำนวนสไลด์ที่ต้องการ (เช่น "10 slides")
- ขอให้ใช้ template แนะนำ (เช่น "Tech startup template")

**ถ้าทำเอง:**
- ใช้ screenshot จริงของแอปจะดูเชื่อถือมากกว่า
- เพิ่ม branding elements (logo, color scheme)
- เพิ่ม testimonial ถ้ามีลูกค้าใช้จริง
